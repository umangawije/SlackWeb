User Experience Design(UX)

UX design considers the elements that shape a users experience with a product or service, how these elements make the user feel, and how easy it is for the user to accomplish their desired tasks.

User Design Experience(UI)

User interface (UI) design refers to the visual design of a products digital interface, such as an app screen or website. Its the process of designing the visual and interactive properties of a visual experience.
